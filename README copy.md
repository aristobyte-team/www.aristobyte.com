## 🚀 Installation & Setup

1. Clone the repository:

```sh
git clone https://github.com/dhunanyan/aristobyte.git
cd aristobyte
```

2. Install dependencies:

```sh
npm install
# OR
yarn install
```

3. Run the project locally:

```sh
npm dev
# OR
yarn dev
```
